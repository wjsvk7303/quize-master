## 🎮 게임 소개

Quiz Master는 한국사, 과학, 지리, 예술과 문화 등 다양한 분야의 상식 문제를 제공하는 웹 기반 퀴즈 게임입니다.
깔끔하고 현대적인 UI/UX로 누구나 쉽게 즐길 수 있습니다.

### 주요 특징
- ✨ **프리미엄 디자인**: 그라데이션 배경, 애니메이션, 글래스모피즘 효과
- 📱 **완벽한 반응형**: 모바일, 태블릿, 데스크톱 모두 지원
- 🎯 **즉시 피드백**: 정답/오답을 실시간으로 확인
- 📊 **상세한 통계**: 카테고리별 정답률 분석
- 🏆 **순위 시스템**: LocalStorage를 활용한 영구 기록

## ⚡ 주요 기능

### 1. 시작 화면
- 이름 입력 기능
- 5가지 카테고리 선택 (전체, 한국사, 과학, 지리, 예술과 문화)
- 카테고리별 문제 개수 표시
- 인터랙티브 카드 UI

### 2. 퀴즈 화면
- **플레이어 정보**: 아바타 + 이름 표시
- **실시간 점수**: 현재 점수 실시간 업데이트
- **진행률 바**: 시각적 진행도 표시 (Shimmer 애니메이션)
- **문제 표시**: 카테고리 배지 + 질문
- **4지선다 옵션**: 호버 효과가 있는 선택지
- **즉시 피드백**:
  - ✅ 정답: 녹색 배경 + 펄스 애니메이션
  - ❌ 오답: 빨간색 배경 + 흔들림 애니메이션
- **해설 박스**: 각 문제마다 상세한 해설 제공

### 3. 결과 화면
- **원형 진행률 링**: SVG 애니메이션으로 점수 시각화
- **등급 배지**: S/A/B/C/D/F 등급 표시
- **최종 점수**: 큰 글씨로 강조
- **정답/오답 개수**: 정확한 통계
- **카테고리별 정답률**: 색상 코드로 한눈에 파악
- **격려/축하 메시지**: 등급에 따른 맞춤 메시지

### 4. 순위 시스템
- **영구 저장**: LocalStorage 활용
- **상위 10개 기록**: 점수 내림차순 정렬
- **상세 정보**:
  - 순위 (1-3위는 메달 이모지)
  - 플레이어 이름
  - 날짜 및 시간
  - 정답률
  - 카테고리
  - 점수
  - 등급
- **특별 디자인**: 1-3위는 금/은/동 배경
- **순위 초기화**: 모든 기록 삭제 가능

### 5. 추가 기능
- **다시 도전**: 동일 카테고리로 재시작
- **홈으로**: 시작 화면으로 이동
- **순위 보기**: 명예의 전당 확인

## 🛠 기술 스택

### 데이터

- **카테고리**: 한국사, 과학, 지리, 예술과 문화

## 🚀 실행 방법

### 1. 프로젝트 다운로드
```bash
git clone <repository-url>
cd 3.test
```

### 2. 실행
**방법 1: 직접 실행**
- `index.html` 파일을 더블클릭
- 또는 브라우저에서 직접 열기

**방법 2: Live Server 사용 (권장)**
```bash
# VS Code의 Live Server 확장 설치 후
# index.html에서 우클릭 → "Open with Live Server"
```

**방법 3: Python 서버**
```bash
# Python 3
python -m http.server 8000

# 브라우저에서 http://localhost:8000 접속
```

### 3. 게임 시작
1. 이름 입력
2. 카테고리 선택
3. "시작하기" 버튼 클릭
4. 퀴즈 풀기!

## 📖 게임 규칙

### 점수 계산
- **정답**: +10점
- **오답**: 0점
- **최대 점수**:
  - 전체 (40문제): 400점
  - 개별 카테고리 (10문제): 100점

### 진행 방식
1. 문제를 읽고 4개의 선택지 중 하나를 선택
2. 즉시 정답/오답 피드백 확인
3. 해설을 읽고 "다음 문제" 버튼 클릭
4. 모든 문제를 풀면 결과 화면으로 이동

### 제한 사항
- ⏱ **시간 제한 없음**: 천천히 생각하고 답변 가능
- 🔄 **답변 변경 불가**: 선택한 답은 변경 불가
- 📝 **순차 진행**: 문제를 건너뛸 수 없음

## 🏆 등급 시스템

점수 범위에 따라 6개 등급으로 분류됩니다:

| 등급 | 점수 범위 | 칭호 | 이모지 |
|------|-----------|------|--------|
| **S** | 360-400점 | 상식 박사 | 🏆 |
| **A** | 320-359점 | 상식 고수 | 🌟 |
| **B** | 280-319점 | 상식 우수 | 🎉 |
| **C** | 240-279점 | 상식 보통 | 👍 |
| **D** | 200-239점 | 상식 부족 | 📚 |
| **F** | 0-199점 | 더 노력 필요 | 💪 |

```
3.test/
├── index.html          # 메인 HTML 파일

├── data/

```

### 파일 설명

#### `index.html`
- 시작 화면, 퀴즈 화면, 결과 화면, 순위 모달
- 시맨틱 HTML5 구조
- Noto Sans KR 폰트 로드

#### `style.css`
- CSS Variables로 일관된 디자인 유지
- 반응형 미디어 쿼리 (768px, 480px)
- 애니메이션 및 트랜지션 효과
- 모달 스타일링

#### `script.js`
- **문제 데이터**: QUESTIONS_DATA 배열 (40문제)
- **게임 로직**:
  - 문제 표시 및 답안 체크
  - 점수 계산
  - 진행률 관리
- **순위 시스템**:
  - saveRanking(): 결과 저장
  - getRankings(): 순위 조회
  - showRankingModal(): 순위 표시
  - clearRanking(): 순위 초기화
- **유틸리티**:
  - shuffleArray(): Fisher-Yates 셔플
  - getGrade(): 등급 계산
  - animateScoreRing(): SVG 애니메이션
